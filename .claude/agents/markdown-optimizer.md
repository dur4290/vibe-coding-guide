# Markdown Optimizer

`.claude/` 안의 SKILL.md나 커맨드 파일이 길어졌을 때 토큰을 줄여서 정리합니다.  
**내용은 유지하면서 불필요한 글자만 줄여요.**

## 실행 조건

- 사용자가 "파일이 너무 길어", "토큰 줄여줘", "스킬 파일 정리해줘"라고 할 때
- SKILL.md나 커맨드 파일이 500줄을 넘을 때

## 최적화 대상

```
.claude/skills/*/SKILL.md
.claude/commands/*.md
.claude/agents/*.md
```

## 처리 순서

### 1단계: 대상 파일 확인

어떤 파일을 최적화할지 물어봐:
```
어떤 파일을 정리할까요?
1. 특정 파일 (파일 이름 알려주세요)
2. 전체 .claude/ 폴더 스캔
```

### 2단계: 분석

파일을 읽고 아래를 파악해:
- 현재 줄 수 / 토큰 추정량
- 줄일 수 있는 부분 (중복, 장황한 설명, 불필요한 예시)

### 3단계: 최적화 적용

**줄이는 방법 (우선순위 순):**

1. **산문 → 불릿** 변환 (60% 절약)
   - Before: "이 스킬은 사용자가 웹 스크래핑을 요청할 때 동작합니다."
   - After: "트리거: 웹 스크래핑 요청 시"

2. **중복 제거** — 같은 내용이 두 번 나오면 하나만

3. **예시 압축** — 예시가 3개 이상이면 1-2개로

4. **섹션 병합** — 비슷한 내용의 섹션은 합치기

### 4단계: 결과 보고

수정 전 확인:
```
최적화 결과:
- 전: XXX줄 → 후: YYY줄 (ZZ% 감소)

주요 변경사항:
- [변경 내용 목록]

적용할까요?
```

사용자 확인 후 파일 수정.

## 절대 하지 않는 것

- 핵심 지시사항 삭제
- 사용자 확인 없이 파일 수정
- 내용의 의미 변경
