# /idea — 아이디어 메모

내가 떠오른 아이디어를 빠르게 메모로 남깁니다.

## 사용법

```
/idea [아이디어 제목 또는 내용]
```

예시:
```
/idea 유튜브 자막 자동 저장 스크립트
/idea 매일 날씨를 카카오톡으로 받아보는 자동화
```

---

## 실행 순서

학생에게 터미널 명령을 직접 입력하라고 시키지 말고, Claude가 폴더 확인과 파일 생성을 직접 처리해.

1. 오늘 날짜(KST 기준)를 확인해
2. `notes/ideas/` 폴더가 없으면 만들어
3. 파일 이름: `YYYY-MM-DD-아이디어제목.md`
   - 제목에서 공백은 하이픈으로, 특수문자는 제거
4. 아래 템플릿으로 파일을 생성해
5. 저장 경로를 알려줘

---

## 파일 템플릿

```markdown
---
date: YYYY-MM-DD
status: 씨앗
---

# 아이디어 제목

## 한 줄 설명
(사용자가 말한 내용 그대로)

## 왜 만들고 싶은가?


## 어떻게 만들 수 있을까?


## 참고할 것


---
*status: 씨앗 → 자라는 중 → 완성 → 보관*
```

---

## 주의사항

- 아이디어 제목이 없으면 먼저 물어봐줘
- 이미 비슷한 아이디어 파일이 있으면 알려줘 (새로 만들지 말고)
- 파일 생성 후 바로 열어서 내용 추가를 도와줘도 돼
