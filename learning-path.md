# 바이브코딩 학습 로드맵

가이드북을 끝낸 뒤에도 이 워크스페이스를 계속 사용할 수 있습니다.  
아래 단계는 순서대로만 가야 하는 시험표가 아니라, 지금 필요한 프로젝트를 고르는 지도입니다.

---

## Level 1. 입문

목표: Claude Code와 함께 파일을 만들고, 실행하고, 수정하고, 저장하는 흐름에 익숙해집니다.

추천 프로젝트:
- 마크다운 보고서 만들기
- 테스트 폴더 파일 정리 자동화
- 아이디어 메모와 배움 기록 정리

Claude에게 요청할 말:

```text
가이드북에서 배운 흐름을 복습하고 싶어.
작은 파일 정리 자동화 프로젝트를 하나 더 추천해줘.
```

---

## Level 2. 자동화 기초

목표: 반복 작업을 작은 Python 스크립트로 바꿉니다.

추천 프로젝트:
- CSV 파일 열 정리하기
- 여러 텍스트 파일을 하나로 합치기
- 파일명에 날짜 붙이기
- 폴더 안 파일 목록을 엑셀/CSV로 저장하기

Claude에게 요청할 말:

```text
Level 2 자동화 프로젝트를 해보고 싶어.
내가 자주 하는 반복 작업을 물어보고, projects/ 안에 새 프로젝트 계획서를 만들어줘.
```

---

## Level 3. 자동화 고급

목표: 실제 자료를 다루되, 위험한 파일 이동이나 삭제는 미리보기와 확인 단계를 넣어 안전하게 처리합니다.

추천 프로젝트:
- 다운로드 폴더 정리 고급 버전
- 같은 이름 파일 자동 번호 붙이기
- 날짜별 결과 폴더 만들기
- 실행 전 미리보기와 확인 기능 추가
- 웹페이지에서 공개 자료를 가져와 CSV로 저장하기

Claude에게 요청할 말:

```text
자동화 프로젝트를 조금 더 고급으로 만들고 싶어.
실행 전 미리보기, 확인 후 실행, 오류 메시지 안내가 들어간 프로젝트를 설계해줘.
```

---

## Level 4. 문서와 발표 만들기

목표: Claude Code로 보고서, 발표자료, HTML 문서를 만들며 결과물을 보기 좋게 정리합니다.

추천 프로젝트:
- HTML 보고서 만들기
- HTML로 발표 슬라이드 만들기
- 마크다운 내용을 발표 대본으로 바꾸기
- 조사 내용을 표와 섹션으로 정리한 문서 만들기

Claude에게 요청할 말:

```text
HTML로 발표 슬라이드를 만들어보고 싶어.
입력은 마크다운 목차로 받고, 결과는 브라우저에서 열 수 있는 slides.html로 만드는 프로젝트를 설계해줘.
```

---

## Level 5. 앱과 대시보드

목표: 단일 스크립트를 넘어, 입력 화면이나 결과 화면이 있는 작은 도구를 만듭니다.

추천 프로젝트:
- Streamlit 데이터 대시보드
- CSV 업로드 후 요약표 만들기
- 검색 가능한 메모 페이지
- 공모전 데이터 탐색 도구

Claude에게 요청할 말:

```text
작은 웹앱이나 대시보드를 만들어보고 싶어.
너무 복잡하지 않게, 로컬에서 실행되는 첫 프로젝트를 추천해줘.
```

---

## Level 6. 나만의 작업 시스템

목표: 자주 하는 일을 커맨드, 스킬, 템플릿으로 정리해서 내 워크스페이스에 흡수합니다.

추천 프로젝트:
- 자주 쓰는 보고서 템플릿 만들기
- 반복 프로젝트용 README 템플릿 만들기
- 나만의 슬래시 커맨드 만들기
- 특정 작업용 스킬 만들기

Claude에게 요청할 말:

```text
내가 자주 하는 작업을 이 워크스페이스에 커맨드나 템플릿으로 저장하고 싶어.
먼저 어떤 작업을 자동화하면 좋을지 질문해줘.
```

---

## 막힐 때

어떤 단계인지 몰라도 괜찮습니다. Claude에게 이렇게 말하세요.

```text
현재 워크스페이스 상태를 확인하고,
내가 진행 중인 프로젝트와 다음에 할 일 하나를 알려줘.
```

저장하고 싶을 때:

```text
/save 오늘 작업 저장
```

오늘 배운 것을 남기고 싶을 때:

```text
/daily
```
